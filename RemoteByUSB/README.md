## Сборка робота

Необходимо удостовериться, что к роботу подключены моторы следующим образом:

1. Левый мотор подключен к выходу расширительного модуля №1
2. Правый мотор подключен к выходу расширительного модуля №2

## Прошивка робота

Matrix Mini основан на плате Arduino Uno R3.
Прошивается плата через программу [Arduino IDE](https://arduino.cc)

Необходимо установить программу, выбрать необходимые параметры платы.

Также необходимо установить библиотеку позволяющие плате управлять роботом:

- Matrix Controller by KKITC
- Matrix Mini Library by KKITC

Скачать исходный код прошивки ["remote_by_usb.ino"](ArduinoCode/remote_by_usb.ino) и загрузить его на плату.

## Как открыть список устройств на Windows

1. Нажмите `Win + X` и выберите **Диспетчер устройств**
    1. Или нажмите `Win + R`, введите `devmgmt.msc` и нажмите Enter
2. Открыть раздел COM портов
3. Найти микроконтроллер
4. При использовании python укажите номер COM порта в файле ["main.py"](PythonCode/main.py)

## Установка библиотек

1. Нажмите `Win + R`, введите `cmd` и нажмите Enter
2. Введите `pip install pyserial pygame` и нажмите Enter
3. Дождитесь установки необходимых библиотек
4. Закройте окно командной строки
